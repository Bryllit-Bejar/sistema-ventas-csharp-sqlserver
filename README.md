# Sistema de Gestión de Ventas
Aplicación de escritorio desarrollada en C# con SQL Server.

## Funcionalidades
- Registro de clientes
- Registro de productos
- Registro de ventas
- Conexión a base de datos
- Consultas SQL

## Tecnologías
- CSharp
- .NET (Windows Forms)
- SQL Server
